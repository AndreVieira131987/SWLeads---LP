# SWLeads - Landing Page Premium

🚀 **SWLeads** é uma landing page de alta conversão, desenvolvida com tecnologias de ponta para oferecer uma experiência de usuário fluida, moderna e visualmente impactante.

Este projeto utiliza **React 19**, **Vite** e **Tailwind CSS v4** para garantir performance excepcional e design responsivo.

---

## ✨ Funcionalidades

- 📱 **Design Responsivo**: Totalmente adaptado para dispositivos móveis, tablets e desktop.
- 🎨 **Interface Moderna**: Estética premium com componentes interativos e animações suaves.
- ⚡ **Performance Otimizada**: Carregamento ultra-rápido graças ao Vite e componentes leves.
- 🧩 **Componentes Reutilizáveis**: Estrutura modular para fácil manutenção e escalabilidade.
- 🤖 **Integração AI**: Pronto para integração com Gemini AI da Google.
- 🔍 **SEO Friendly**: Estrutura semântica para melhor ranqueamento em motores de busca.

---

## 🚀 Como Executar

Siga os passos abaixo para iniciar o projeto localmente:

1. **Clone o repositório** (ou baixe os arquivos):
   ```bash
   git clone <url-do-repositorio>
   cd swleads-lp
   ```

2. **Instale as dependências**:
   ```bash
   npm install
   ```

3. **Configure as Variáveis de Ambiente**:
   Crie um arquivo `.env.local` na raiz do projeto e adicione sua chave da API do Gemini (se necessário para as funcionalidades de IA):
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```

4. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

5. **Acesse no navegador**:
   O Vite abrirá automaticamente na porta `3000` (conforme configurado no `package.json`). Caso contrário, acesse: `http://localhost:3000`
